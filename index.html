Korpkvädet - Hobby project website

A website for a LARP (live role-playing game) project I was involved in, one of my tasks being to setup the project website. The website is pretty straightforward, built with Wordpress, to allow for other team members to easily add content, and using Airtable to store participant registration data. The work consisted mainly of implementing the layout and structure and navigation, then styling according to specifications, and adding plugins and tweaking functionality for things like sitemap, search, changelog, table of contents, dynamic registration form (not available anymore as registration has closed) and management of registration data (storing it in Airtable).
